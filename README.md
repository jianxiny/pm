启动

### Client

```
cd client && npm i && npm run dev
```

### Server

```
cd server
python3 -m venv envv(你可以指定的名字)
source ./envv/bin/activate
pip install flask

export FLASK_APP = app
export FLASK_ENV = development
flask run
```
